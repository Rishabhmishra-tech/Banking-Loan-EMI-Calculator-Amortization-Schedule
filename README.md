# Banking Loan EMI Calculator & Amortization Schedule

## 📌 Project Overview

The **Banking Loan EMI Calculator & Amortization Schedule** is an Excel-based analytics project that transforms raw banking loan data into meaningful insights. It includes data cleaning, loan portfolio analysis using Pivot Tables and Charts, an interactive EMI calculator, and a dynamic month-by-month amortization schedule.

This project demonstrates practical Excel skills used in banking, finance, and data analytics.

---

## 🎯 Project Objectives

- Import and organize banking loan data.
- Clean and standardize raw data using Excel functions.
- Analyze loan portfolios using Pivot Tables and Pivot Charts.
- Calculate EMI automatically based on loan details.
- Generate a complete amortization schedule.
- Provide interactive loan analysis through dashboards and slicers.

---

## 📂 Project Workflow

### Step 1: Data Collection

- Downloaded Banking Loan Dataset from Kaggle.
- Imported the CSV dataset into Microsoft Excel.

The dataset contains:

- Loan ID
- Customer Name
- Loan Type
- Loan Amount
- Interest Rate
- Loan Tenure
- Branch
- Application Date

---

### Step 2: Data Cleaning

Data was cleaned using built-in Excel functions.

| Function | Purpose |
|----------|----------|
| TRIM() | Removes unnecessary spaces |
| PROPER() | Converts names into proper case |
| UPPER() | Standardizes text values |
| ISBLANK() | Identifies missing values |

After cleaning:

- Standardized customer names
- Removed extra spaces
- Corrected text formatting
- Validated missing data

---

### Step 3: Data Analysis

Created Pivot Tables to analyze:

- Loan Amount by Loan Type
- Loan Count by Branch
- Total EMI
- Total Interest

Created Pivot Charts for visual representation.

---

### Step 4: EMI Calculator

Developed a dynamic EMI Calculator where users can:

- Search customer records
- Select Loan ID
- Automatically load loan information
- Calculate EMI instantly

The calculator computes:

- Monthly EMI
- Total Interest
- Total Payment

---

### Step 5: Amortization Schedule

Generated a complete repayment schedule showing:

- Month Number
- Payment Date
- Opening Balance
- EMI
- Principal Paid
- Interest Paid
- Closing Balance

The schedule updates automatically whenever loan inputs change.
